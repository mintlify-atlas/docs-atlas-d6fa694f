PinchTab offers multiple installation methods to fit your workflow. Choose the method that works best for your environment.

## Quick Installation

<Steps>
  <Step title="Choose your installation method">
    Select from curl script (recommended), npm, Docker, or build from source.
  </Step>
  <Step title="Install PinchTab">
    Follow the instructions for your chosen method below.
  </Step>
  <Step title="Verify installation">
    Run `pinchtab --version` to confirm installation.
  </Step>
</Steps>

## Installation Methods

<Tabs>
  <Tab title="Curl Script (Recommended)">
    The quickest way to get started on macOS and Linux.

    ```bash
    curl -fsSL https://pinchtab.com/install.sh | bash
    ```

    **What it does:**
    - Detects your operating system (macOS/Linux)
    - Checks for Node.js 18+ and npm
    - Installs the `pinchtab` npm package globally
    - Verifies the installation

    **Verify installation:**
    ```bash
    pinchtab --version
    ```

    <Note>
      The install script requires Node.js 18+ to be installed on your system.
    </Note>
  </Tab>

  <Tab title="npm">
    Install PinchTab as an npm package for Node.js projects or global CLI access.

    **Global installation (CLI):**
    ```bash
    npm install -g pinchtab
    ```

    **Local installation (SDK):**
    ```bash
    npm install pinchtab
    ```

    **Verify:**
    ```bash
    pinchtab --version
    ```

    [Full npm installation guide →](/installation/npm)
  </Tab>

  <Tab title="Docker">
    Run PinchTab in a container with bundled Chromium.

    **Pull and run:**
    ```bash
    docker run -d -p 9867:9867 pinchtab/pinchtab
    ```

    **Verify:**
    ```bash
    curl http://localhost:9867/health
    ```

    [Full Docker installation guide →](/installation/docker)
  </Tab>

  <Tab title="Build from Source">
    Build PinchTab from source code using Go.

    **Requirements:** Go 1.25+, Git, Chrome/Chromium

    ```bash
    git clone https://github.com/pinchtab/pinchtab.git
    cd pinchtab
    go build -o pinchtab ./cmd/pinchtab
    ./pinchtab --version
    ```

    [Full build guide →](/installation/build-from-source)
  </Tab>
</Tabs>

## System Requirements

### Operating Systems

- **macOS** (Intel and Apple Silicon)
- **Linux** (amd64, arm64)
- **Windows** (via WSL2)

### Dependencies

PinchTab requires a Chrome or Chromium browser to be installed:

<CodeGroup>
```bash macOS
brew install chromium
```

```bash Ubuntu/Debian
sudo apt install chromium-browser
```

```bash RHEL/CentOS
sudo yum install chromium
```
</CodeGroup>

<Note>
  Docker installations include Chromium bundled in the container.
</Note>

## Quick Start

Once installed, start the PinchTab server:

```bash
pinchtab
```

**Expected output:**
```
🦀 Pinchtab Dashboard port=9867
dashboard ready url=http://localhost:9867/dashboard
```

The server is now running on `http://localhost:9867`. Test it:

```bash
curl http://localhost:9867/health
```

## Troubleshooting

### "command not found: pinchtab"

**npm users:**
Add npm's global bin directory to your PATH:

```bash
export PATH="$(npm config get prefix)/bin:$PATH"
# Add to ~/.bashrc or ~/.zshrc to persist
```

### "Chrome not found"

Install Chrome or Chromium, or specify a custom path:

```bash
CHROME_BIN=/path/to/chrome pinchtab
```

### Permission errors (npm)

If you get permission errors during global npm install:

**Option 1: Use nvm (recommended)**
```bash
curl https://github.com/nvm-sh/nvm/raw/master/install.sh | bash
nvm install node
npm install -g pinchtab
```

**Option 2: Use user prefix**
```bash
npm install -g --prefix "$HOME/.local" pinchtab
export PATH="$HOME/.local/bin:$PATH"
```

### Port already in use

Change the default port (9867):

```bash
BRIDGE_PORT=9868 pinchtab
```
